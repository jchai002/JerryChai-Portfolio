HippoPath
Professional Project
{Click for Source Code}

This is a web application that helps medical school students to find carpools for hospital interviews. The majority of the front-end is built using ReactJs, with the data being fed from a rails back-end. The major features include the ability to post interviews, search interviews, and message other students upon finding a match using a real time chat feature similar to facebook messenger.

Dreamland
Professional Project
{Click for Source Code}

This is the official business website for Dreamland Mattress, where I was an I.T. consultant. It was built with the static page serving framework Jekyll, and deployed on AWS s3.
