Coffee Friends

 {Click for Source Code}

A web application that allows people to buy coffee for friends. There is also a built in charity feature that let people buy coffee for strangers who may not be able to afford it. This application was developed with a team of four in six days using the Rails framework.

Dreamland

 {Click for Source Code}

This is the official business website for Dreamland Mattress, where I was an I.T. consultant. It was built with the static page serving framework Jekyll, and deployed on AWS s3. Because many of Dreamland's customers are Chinese, I designed this website with bilingual support. The content can be switched freely between English and Chinese by clicking on the language option on the navigation bar.

Edelweiss Music

 {Click for Source Code}

This is the official website for the band Edelweiss. It gives full CRUD access to the band members, allowing them to upload and manage their media content. It integrates with the MailChimp API for mailing list management. This is a solo project built on the Rails framework.

Lunch Roulette

 {Click for Source Code}

This app takes away the decision fatigue from choosing a lunch spot. Users only have to pick the distance they are willing to travel and a minimum yelp rating, and the app will select a random restaurant that fits the criteria. This app uses the yelp api and the google maps api.
